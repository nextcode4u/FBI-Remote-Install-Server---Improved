# FBI Remote Install Enhanced (v4)

A **simple, colorful helper** for installing CIAs over the network using  
**FBI → Remote Install → Receive URLs over the network**.

Designed to be **more reliable** and **less annoying** than the stock workflow.

---

## Features

- **IP history**
  - Enter your 3DS IP once
  - Next runs let you **pick from a list**
- Drag & drop a CIA **or a folder**
- Batch installs
- Live progress:
  - % complete
  - MB/s
  - ETA
  - Overall batch progress
- Hotkeys while running:
  - **R** → re-send URLs
  - **Q** → quit
- Colorful terminal output
- Automatic log file for every run

---

## Requirements

- Python 3.7+
- FBI installed on your 3DS
- PC and 3DS on the same network
